3D Data Visualization Engine

Developed a modern, user-friendly, and high-performance 3D Data Visualization Engine capable of visualizing seismic, well, and surface data used in the petroleum sector. Utilizing Three.js, the visualization engine features a dynamic 3D coordinate system to represent complex geological data interactively. Additionally, specialized algorithms were implemented to handle large-scale datasets efficiently.

Active Object Tracking On a Drone" was my final year project at university. I developed this project on the Airsim simulator using Python and AI for active object tracking.
